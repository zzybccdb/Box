# Boxplot

#### 利用 pixi ＋ d3 实作的　boxplot
#### 根据使用者传入资料绘制出 Min, Q1, Median, Q3, Max
- 范例如下
``` JavaScript
      data:{
        'test1':[1,3,5,7,9],
        'test2':[10,20,55,80,100]
      }
```

- ![](https://i.imgur.com/YwzNjZK.png)
- 鼠标移动至图表,显示具体数值
